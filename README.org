This repository hosts [[http://srfi.schemers.org/srfi-77/][SRFI 77]]: Preliminary Proposal for R6RS Arithmetic, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-77/srfi-77.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-77/][join the SRFI-77 mailing list]] and send your message there.
